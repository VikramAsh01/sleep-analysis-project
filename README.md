# 💤 Sleep Analysis Project

## 📌 Overview

This project analyzes the impact of lifestyle factors such as stress, screen time, caffeine, and alcohol on sleep quality and cognitive performance.

---

## 🎯 Objectives

1. **Lifestyle Impact Analysis**
   Analyze how stress, screen time, caffeine, and alcohol affect sleep quality using scatter plots.

2. **Regression Analysis**
   Study the relationship between sleep quality and cognitive performance.

3. **Hypothesis Testing (Stress vs Cognitive Performance)**
   Perform a t-test to determine whether stress significantly affects cognitive performance.

4. **Country & Occupation Analysis**
   Analyze sleep duration patterns across countries and occupations.

5. **Hypothesis Testing (Stress vs Sleep Quality)**
   Test whether stress significantly impacts sleep quality.

---

## 🛠️ Technologies Used

* Python
* Pandas
* Seaborn
* Matplotlib
* Scikit-learn
* SciPy

---

## 📊 Key Insights

* Higher stress levels are associated with lower sleep quality.
* Sleep quality positively influences cognitive performance.
* Lifestyle factors like screen time and caffeine show noticeable trends affecting sleep.
